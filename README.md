### Modrinth Gallery Text Background

Sometimes when looking at images on Modrinth, I can't read the text that appears at the bottom because the background is transparent. As a solution, you could try coloring the background.

#### 🧩 How it works
When opening a gallery section and clicking any of images, it add a background to text that appears on bottom of image.

#### ⚙️ Main features
- Makes texts that appear on bottom of images to more readeble.

#### 🔒 Privacy and safety
- The script does **not** collect or send any user data.  
- It contains **no tracking, ads, or hidden requests**.  
- All operations run locally in your browser on the Modrinth website.

#### 💡 Notes
- Works only on Modrinth's Gallery page.
